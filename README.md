# aggna-cli

> 阿格娜(aggna)，一个轻量的脚手架，用于方便地获取所需开发模板！

*目前只支持 Electron-vue 模板*

## 安装
* `yarn add aggna-cli -g`

## 使用
* `aggna init [project_name]`

## 模板
[aggna-electron-template](https://github.com/zonghua2016/aggna-electron-template.git)
